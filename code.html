The Code for REWIND

                    
                        //Get user input
                        //Controller Function
                        function getString() {
                            document.getElementById("alert").classList.add("invisible");
                            let userString = document.getElementById("revString").value;
                            let revString = reverseString(userString);
                            displayString(revString);
                        }

                        //Reverse text in array
                        //Logic Function
                        function reverseString(userString) {
                            let revString = [];
                            for (let i = userString.length - 1; i >= 0; i--) {
                                revString += userString[i];
                            }
                            return revString;
                        }

                        //Display text on page
                        //Display Function
                        function displayString(revString) {
                            document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
                            document.getElementById("alert").classList.remove("invisible");
                        }

The Code is structured in three functions.

Get String

Get String is the Controller function which grabs the user input, hands it off to Reverse String and gets back an array which it sends to Display String.

Reverse String

Reverse String is the Logic function which takes in the variable of user input, runs a for loop to reverse it into and array and sends it back to Get String.

Display String

Display String is the Display function which takes in the array created by Reverse String, manipulates the DOM to add the message on the page and removes the Class of 'invisible' so the message will show on the page.
